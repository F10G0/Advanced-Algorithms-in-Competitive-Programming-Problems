# Archive — Additional Course Problems

This directory contains additional problems from the course, preserved to maintain a complete collection within the repository.

All problems here include full implementations with input/output handling and test scripts, but typically have lighter documentation or fewer annotated materials compared to the main problem set.

---

## Structure

Each subdirectory follows a consistent layout across the repository:

- `problem/` — original problem statement  
- `src/` — final implementation  
- `tests/` — sample inputs, outputs, and test scripts  

---

## Problem Overview

A brief classification of each archived problem:

### `tab`
- Topic: number theory / linear Diophantine equations  
- Focus: solving congruence alignment via extended Euclidean algorithm  

### `tes`
- Topic: greedy / net balance reduction  
- Focus: transforming edge-based debts into node-level matching  

### `toy`
- Topic: greedy / filtering  
- Focus: feasibility check followed by minimum selection  

### `zoo`
- Topic: binary search / greedy  
- Focus: parametric search with monotonic feasibility check  

---

## Notes

- These problems are **fully correct and complete**, but are:
  - generally simpler, or  
  - less documentation-heavy, or  
  - less representative of multi-stage algorithm design  

- The main repository directories contain the more **in-depth, documented, and showcase-oriented** solutions.

- This archive ensures that **all course problems are retained in a consistent, runnable, and reproducible format**.
